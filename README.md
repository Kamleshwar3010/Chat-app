## üõ†Ô∏è Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Kamleshwar3010/Chat-app.git
    cd Chat-app
    ```

2.  **Install Dependencies**
    Navigate to the backend directory and install the required packages.
    ```bash
    cd back-end
    npm install
    ```

3.  **Set up MongoDB**
    Ensure you have MongoDB installed and running locally on `mongodb://localhost:27017/register`.

4.  **Run the Server**
    ```bash
    npm start
    ```
    The server will start on `http://localhost:3000`.

5.  **Open in Browser**
    Visit `http://localhost:3000` to start chatting!
